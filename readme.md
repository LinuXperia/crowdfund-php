## Installation

Instructions to Install:
-------------------------------------

1. Clone to your environment
2. Rename `.env.example` to `.env` file. Place your credentials there.
3. Run `composer update`
4. Run `gulp`
5. Run `php artisan migrate`
6. Run `php artisan db:seed`

## To-Do After Install
